# BASKET TOURNAMENT

## GESTION DES TOURNOIS

### Install

#### Node

Run : `npm install`

### Run

`npm run server`

`npm run client`

### Database

#### Docker

`docker-compose build`
`docker-compose up -d`
`docker-compose down`

`docker exec -it basketdb-docker bash`
`docker exec -it basketdb-docker mysql -u root -p`
